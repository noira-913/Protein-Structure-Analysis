## 1. 프로젝트 개요

이 프로젝트는 **ALMA (Atomic-Level Molecular Analysis)** 라는 이름의 단백질 구조 분석 엔진입니다.  
핵심 목표는 PDB (Protein Data Bank) 파일로 읽어들인 단백질의:

- **에너지 최소 구조** 탐색
- **구조적 앙상블** 생성 (Monte Carlo 샘플링)
- **고유 무질서 영역** (Intrinsically Disordered Region) 예측
- **구조 간 RMSD 비교**

를 수행하는 것입니다.

### 기술 스택 요약

| 레이어 | 기술 |
|--------|------|
| GUI 및 파이프라인 | Python (PyQt 계열) — `python/gui_main.py` |
| 물리 엔진 (CPU) | C++ + pybind11 — `src/physics_engine.cpp` |
| 물리 엔진 (GPU) | CUDA C++ + pybind11 — `src/physics_engine_cuda.cu` |
| 력장 파라미터 | Python — `python/amber_params.py` |
| 무질서 예측 | Python — `python/iupred.py` |
| 빌드 시스템 | CMake — `CMakeLists.txt` |

## 3. 력장 (Force Field) — AMBER ff14SB

### 력장이란?

**력장(Force Field)** 은 원자들 사이의 상호작용을 수학적으로 기술하는 파라미터 집합입니다.  
이 프로젝트는 AMBER ff14SB 력장을 사용합니다.

> Maier, J.A. et al. (2015) *J. Chem. Theory Comput.* **11**, 3696–3713

### 원자 타입 시스템

AMBER는 원자를 원소만으로 구분하지 않고, **화학 환경**에 따라 세밀하게 분류합니다:

| AMBER 타입 | 의미 |
|------------|------|
| `CX` | Cα (알파 탄소, 척추 뼈대) |
| `2C` | sp³ 탄소 (CH₂ 사슬) |
| `3C` | sp³ 탄소 (CH₃ 말단) |
| `CA` | 방향족 탄소 (페닐기 등) |
| `CT` | 지방족 sp³ 탄소 |
| `N` | 펩티드 결합 질소 |
| `O` | 카르보닐 산소 |

이 구분이 중요한 이유: 같은 탄소라도 위치에 따라 VDW 반지름과 소수성 상호작용 세기가 다릅니다.

### VDW 파라미터 할당

`(잔기명, 원자명)` 쌍을 키로, `(반지름 Å, ε kcal/mol)` 을 값으로 조회:

```
get_atom_params(resname, atomname) → (radius, epsilon, charge)
```

### RESP 부분 전하

각 원자의 부분 전하(partial charge)는 RESP(Restrained ElectroStatic Potential) 방법으로 유도된 값입니다.  
예시 (백본 원자):

| 원자 | 전하 (e) |
|------|----------|
| N (백본) | −0.4157 |
| C (카르보닐) | +0.5973 |
| O (카르보닐) | −0.5679 |
| Cα | +0.0337 |

이 값들은 모든 20가지 표준 아미노산과 HIS의 세 가지 양성자화 형태(HID/HIE/HIP)에 대해 `amber_params.py`에 표로 정의되어 있습니다.

## 4. 퍼텐셜 에너지 함수

시스템의 **총 포텐셜 에너지**는 다음 항들의 합입니다:

$$E_{\text{total}} = E_{\text{LJ}} + E_{\text{DH}} + E_{\text{GB}} + E_{\text{SASA}} + E_{\text{dihedral}}$$

각 항을 순서대로 설명합니다.

---

### 4.1 레너드-존스 포텐셜 (Lennard-Jones)

**무엇을 모델링하는가?** 원자들 사이의 반데르발스 힘 — 아주 가까우면 강하게 밀치고(Pauli 반발), 적당한 거리에서는 약하게 끌어당깁니다(분산력).

$$E_{\text{LJ}}(i,j) = 4\varepsilon_{ij} \left[ \left(\frac{\sigma_{ij}}{r}\right)^{12} - \left(\frac{\sigma_{ij}}{r}\right)^{6} \right]$$

- $r$ : 두 원자 사이의 거리 (Å)
- $\sigma_{ij} = r_i + r_j$ : 두 원자의 VDW 반지름 합 (충돌 직경)
- $\varepsilon_{ij} = \sqrt{\varepsilon_i \cdot \varepsilon_j}$ : 혼합 규칙 (Lorentz-Berthelot)으로 계산된 우물 깊이

**항의 의미:**
- $(\sigma/r)^{12}$ 항: 두 원자 핵이 겹치지 않도록 하는 **단거리 반발** (Pauli 배타 원리)
- $(\sigma/r)^{6}$ 항: 전자 구름 편극으로 생기는 **런던 분산 인력**
- 에너지 최솟값(우물 바닥): $r = 2^{1/6}\sigma$ 에서 $E = -\varepsilon$

**하드 코어 보호:**  
$r < 0.85\,\sigma$ 이면 LJ 항 대신:

$$E_{\text{hard}} = 10^4 \cdot \left(\frac{\sigma}{r}\right)^{12}$$

으로 처리하여 분모가 0에 가까워질 때 수치 발산을 방지합니다.

---

### 4.2 데바이-휘켈 정전기 (Debye-Hückel)

**무엇을 모델링하는가?** 이온 용액(생리 조건) 속에서 두 전하 원자 사이의 정전기 상호작용.

진공 중 쿨롱 법칙 $E = k_e q_i q_j / r$ 에 두 가지 수정이 가해집니다:

1. **유전 차폐 (Dielectric screening):** 물 분자가 전기장을 약화시킴 → 분모에 $\varepsilon_{\text{water}} = 78.5$
2. **이온 차폐 (Ionic screening):** 주변 이온들이 전기장을 더 약화시킴 → 지수 감쇠 $e^{-\kappa r}$

$$E_{\text{DH}}(i,j) = \frac{332.0636 \cdot q_i q_j}{\varepsilon_{\text{water}} \cdot r} \cdot e^{-\kappa r}$$

- $332.0636$ : 단위 변환 계수 (전하 단위 $e$, 거리 Å → 에너지 kcal/mol)
- $\varepsilon_{\text{water}} = 78.5$ : 물의 유전 상수
- $\kappa = 0.1257\,\text{Å}^{-1}$ : 역 데바이 길이 (생리 이온 농도 ~150 mM에 해당)
- 데바이 길이 $\lambda_D = 1/\kappa \approx 8\,\text{Å}$ : 이 거리 이상에서 전기장이 크게 약해짐

**물리적 의미:**  
같은 부호 전하(예: Asp–Asp, Lys–Lys) → $E > 0$ (반발)  
반대 부호 전하(예: Asp–Lys, 염 다리) → $E < 0$ (인력)

---

### 4.3 일반화 Born 용매화 — HCT 모델

**무엇을 모델링하는가?** 물 속 단백질에서 각 전하 원자가 주변 물 분자를 편극시켜 생기는 **용매화 자유 에너지**.

이것은 가장 정교한 항입니다. 원리를 단계적으로 설명합니다.

#### 4.3.1 Born 이온 모델 (배경 이론)

반지름 $a$인 구형 공동(cavity) 안에 전하 $q$가 있을 때, 유전율 $\varepsilon$인 용매 속 용매화 에너지:

$$G_{\text{Born}} = -\frac{1}{2} \left(\frac{1}{\varepsilon_{\text{prot}}} - \frac{1}{\varepsilon_{\text{water}}}\right) \frac{q^2}{a} \cdot C$$

여기서 $C = 332.0636$ (단위 변환), $\varepsilon_{\text{prot}} = 1$ (단백질 내부).

#### 4.3.2 유효 Born 반지름 (Effective Born Radius)

단백질은 구형 이온이 아니므로, 각 원자 $i$에 대해 주변 원자들이 얼마나 물을 '가리는지'를 계산해 **유효 Born 반지름** $\alpha_i$를 구합니다.

**HCT (Hawkins-Cramer-Truhlar) 모델:**

$$\alpha_i^{-1} = r_i^{-1} - \frac{1}{2} \sum_{j \neq i} I(r_{ij}, r_i, r_j)$$

여기서 $I$는 원자 $j$가 원자 $i$ 주변 용매를 가리는 정도를 나타내는 적분:

$$I(r, r_i, r_j) = \frac{1}{L} - \frac{1}{U} + \frac{r^2 - r_j^2 + r_i^2}{2r \cdot r_i^2} \cdot \frac{\ln(L/U)}{2r}$$
$$L = \max(|r - r_j|,\, r_i), \quad U = r + r_j$$

**물리적 의미:**  
- $\alpha_i$가 작다 → 원자 $i$가 단백질 표면에 노출 → 용매화 에너지 큼
- $\alpha_i$가 크다 → 원자 $i$가 단백질 내부에 묻힘 → 용매화 에너지 작음

#### 4.3.3 GB 쌍별 에너지

원자 쌍 $(i,j)$의 GB 기여:

$$E_{\text{GB}}(i,j) = -\frac{1}{2}\left(\frac{1}{\varepsilon_{\text{prot}}} - \frac{1}{\varepsilon_{\text{water}}}\right) \frac{C \cdot q_i q_j}{f_{\text{GB}}}$$

$$f_{\text{GB}} = \sqrt{r_{ij}^2 + \alpha_i \alpha_j \cdot \exp\!\left(-\frac{r_{ij}^2}{4\alpha_i\alpha_j}\right)}$$

- $r_{ij} \to \infty$ 이면 $f_{\text{GB}} \to r_{ij}$ → 쿨롱 극한 (긴 거리에서 GB = DH와 유사)
- $r_{ij} \to 0$ 이면 $f_{\text{GB}} \to \sqrt{\alpha_i \alpha_j}$ → 자체 에너지(self-energy) 극한

> 참고: Hawkins, Cramer & Truhlar (1995) *J. Phys. Chem.* **99**, 11663

---

### 4.4 비극성 SASA 항

**무엇을 모델링하는가?** 소수성 표면이 물에 노출될 때 드는 비용 (엔트로피 페널티 — 물 분자의 배열 제한).

$$E_{\text{SASA}} = \beta_{\text{SA}} + \gamma_{\text{SA}} \sum_i \max(0,\, SA_i)$$

- $\gamma_{\text{SA}} = 0.00542\,\text{kcal/(mol·Å}^2\text{)}$ : 표면 장력 계수
- $\beta_{\text{SA}} = 0.92\,\text{kcal/mol}$ : 기준 오프셋
- $SA_i$ : 원자 $i$의 용매 접근 가능 표면적 (SASA, Å²)

**SASA 계산 방법 (근사):**  
각 원자 $i$에 대해 구면적 $4\pi(r_i + r_{\text{probe}})^2$에서 인접 원자들이 가리는 구면 캡(cap)을 빼는 방식:

$$h_{ij} = \frac{(r_i + r_{\text{probe}}) + (r_j + r_{\text{probe}}) - r_{ij}}{2(r_i + r_{\text{probe}})}$$

$r_{\text{probe}} = 1.4\,\text{Å}$ (물 분자 반지름).

---

### 4.5 이면각(비틀림) 에너지

**무엇을 모델링하는가?** 단일 공유결합 주위를 회전할 때의 에너지 장벽 — 전자 구름 겹침과 입체 반발로 인해 특정 각도가 선호됩니다.

이면각 $\phi$는 네 원자 $a$-$b$-$c$-$d$ 로 정의됩니다:

$$E_{\text{dih}} = \sum_{\text{결합}} \sum_n \frac{V_n}{2} \left[1 + \cos(n\phi - \gamma_n)\right]$$

- $V_n$ : 장벽 높이 (kcal/mol) — 주기성 $n$에 따라 다름
- $n$ : 푸리에 주기 (보통 2 또는 3)
- $\gamma_n$ : 위상 오프셋

**백본 φ/ψ 파라미터 (Ramachandran 페널티):**

| 결합 종류 | 항 1 | 항 2 |
|-----------|------|------|
| 백본 φ | $V=0.40,\, n=3,\, \gamma=\pi$ | $V=0.29,\, n=2,\, \gamma=\pi$ |
| 백본 ψ | $V=0.40,\, n=3,\, \gamma=0$ | $V=0.29,\, n=2,\, \gamma=\pi$ |

**사이드체인 χ 결합:** 원자 종류(O, S, N, C-C)에 따라 다른 파라미터 적용.

**이면각 계산 (Rodrigues 공식):**

벡터 $\mathbf{b}_1 = \mathbf{r}_b - \mathbf{r}_a$, $\mathbf{b}_2 = \mathbf{r}_c - \mathbf{r}_b$, $\mathbf{b}_3 = \mathbf{r}_d - \mathbf{r}_c$ 일 때:

$$\mathbf{n}_1 = \mathbf{b}_1 \times \mathbf{b}_2, \quad \mathbf{n}_2 = \mathbf{b}_2 \times \mathbf{b}_3$$
$$\phi = \text{atan2}\left(\hat{\mathbf{b}}_2 \cdot (\mathbf{n}_1 \times \mathbf{n}_2),\; \mathbf{n}_1 \cdot \mathbf{n}_2\right)$$

---

### 4.6 공유결합 제외 (1-2/1-3 Exclusions)

직접 결합된 원자 쌍(1-2 쌍)과 결합 하나를 사이에 둔 원자 쌍(1-3 쌍)은 비결합 에너지 계산에서 **제외**됩니다.

**이유:** 이 원자들은 이미 결합 에너지 항(결합 신축, 각도 굽힘)으로 다루어지며, 반데르발스/전하 상호작용이 물리적으로 의미 없습니다.

구현: `BondTopology` 의 `excl` 집합에 모든 1-2, 1-3 원자 쌍 인덱스를 저장하고, 쌍별 에너지 계산 루프에서 이 쌍들을 건너뜁니다.

## 5. 공유결합 위상 그래프 (Bond Topology)

`BondTopology` 클래스는 단백질의 **공유결합 그래프**를 구성합니다.

### 구축 과정

1. 각 아미노산 잔기의 결합 템플릿(AMBER prep 파일 기반, 잔기당 10–25개 결합)을 C++ 코드에 내장
2. 잔기들을 순서대로 연결 (백본 N-C-Cα-C(=O)-N 체인)
3. 전체 공유결합 그래프 $G$ 구성 → 인접 리스트 `adj[]`

### 핵심 데이터 구조

```
BondTopology {
    adj[]        : 각 원자의 이웃 원자 목록 (공유결합)
    excl[]       : 1-2, 1-3 제외 쌍 집합
    rot_bonds[]  : 회전 가능 결합 목록 {(i,j), 종류}
    rot_bond_sides[] : 각 회전 결합의 j-측 원자 집합
    dihedrals[]  : DihRecord 목록 {a,b,c,d, 항들}
}
```

### 회전 가능 결합 판별

결합 $(i,j)$가 회전 가능하려면:
- 환 구조(ring) 내에 있지 않을 것
- 말단 원자가 아닐 것 (회전해도 하류 원자 없음)
- 펩티드 결합(C-N)은 **부분 이중결합**이므로 회전 가능 목록에서 제외

이동 가중치: **50% 백본 φ/ψ, 40% 사이드체인 χ, 10% 기타**

## 6. 몬테카를로 샘플링

### 왜 Monte Carlo인가?

단백질의 에너지 경관(energy landscape)은 매우 복잡합니다:  
$N$개 원자의 구조 공간은 $3N$ 차원이고, 수많은 국소 최솟값(local minima)이 존재합니다.  
그래디언트 하강법(에너지 최소화)은 가장 가까운 국소 최솟값으로만 수렴하므로,
**열적 요동(thermal fluctuation)을 허용하는 MC 샘플링**으로 더 넓은 구조 공간을 탐색합니다.

---

### 6.1 비틀림각 이동 (Torsion MC Move)

**핵심 아이디어:** 원자를 데카르트 좌표로 무작위 이동시키면 결합 길이·각도가 깨집니다.  
대신 **비틀림각(torsion angle)** 을 변경하면 결합 기하학이 항상 유지됩니다.

#### 이동 과정

1. 회전 가능 결합 $(i,j)$ 를 가중치에 따라 무작위 선택
2. 회전각 $\delta\phi$ 샘플링:
   - 백본: $\delta\phi \sim U[-5°, +5°]$
   - 사이드체인: $\delta\phi \sim U[-30°, +30°]$
3. 결합 축 $\hat{\mathbf{u}} = (\mathbf{r}_j - \mathbf{r}_i) / |\mathbf{r}_j - \mathbf{r}_i|$ 에 대해
   $j$-측 원자들을 **Rodrigues 회전 공식**으로 일괄 회전:

$$\mathbf{r}' = \mathbf{r} \cos(\delta\phi) + (\hat{\mathbf{u}} \times \mathbf{r}) \sin(\delta\phi) + \hat{\mathbf{u}}(\hat{\mathbf{u}} \cdot \mathbf{r})(1 - \cos(\delta\phi))$$

4. 에너지 변화 $\Delta E$ 계산 (이동된 원자들과 경계 이면각만 재계산 → $O(N/2)$ 최악 케이스이지만 실제로는 국소적)
5. 메트로폴리스 기준으로 수락/거부

---

### 6.2 메트로폴리스 기준 (Metropolis Criterion)

$$P(\text{수락}) = \begin{cases} 1 & \Delta E \leq 0 \\ e^{-\Delta E / T} & \Delta E > 0 \end{cases}$$

- $T$ : 온도 파라미터 (kcal/mol 단위; 실제 온도와 직접 대응하지는 않음)
  - 기본값 $T = 0.6$ — 약 300 K 생리 온도에 근사
- $\Delta E < 0$ : 에너지가 낮아지는 이동은 **항상** 수락
- $\Delta E > 0$ : 에너지가 높아지는 이동도 $e^{-\Delta E/T}$ 확률로 수락 → 국소 최솟값 탈출 가능

이 기준은 **볼츠만 분포**를 정상 분포(stationary distribution)로 하는 마르코프 체인을 구성합니다:

$$P(\mathbf{x}) \propto e^{-E(\mathbf{x})/T}$$

즉, 충분히 오래 실행하면 에너지가 낮은 구조를 더 자주 샘플링합니다.

---

### 6.3 이웃 목록 (Neighbor List)

모든 원자 쌍을 매 스텝마다 검사하면 $O(N^2)$ 비용이 발생합니다.  
**이웃 목록**은 차단 거리($r_{\text{cut}} = 12\,\text{Å}$) + 스킨 레이어($r_{\text{skin}} = 2\,\text{Å}$) 이내의 쌍만 미리 목록화:

$$r_{\text{NL}} = r_{\text{cut}} + r_{\text{skin}} = 14\,\text{Å}$$

- 매 MC 스텝에서는 저장된 목록만 순회 → $O(k \cdot N)$, $k$ = 평균 이웃 수
- 임의 원자가 $r_{\text{skin}}/2 = 1\,\text{Å}$ 이상 이동하면 목록 재구성
- 재구성 시 최신 GPU 코드에서는 `gpu_total_energy()` 로 전체 에너지를 GPU에서 재계산

## 7. IUPred 무질서 예측

### 고유 무질서 단백질 (IDP)란?

정상적인 생리 조건에서 **안정된 3차 구조를 갖지 않는** 단백질 또는 단백질 영역.  
전사 인자, 신호 전달 단백질, 상전이(phase separation) 허브 단백질에서 광범위하게 나타납니다.

무질서 단백질의 특성:
- 소수성 아미노산이 적음 → 소수성 코어 형성 불가
- 하전된 아미노산(Asp, Glu, Lys, Arg)이 많음 → 정전기 반발
- 글리신(Gly), 프롤린(Pro) 비율이 높음 → 구조적 유연성

---

### 7.1 알고리즘 개요 (IUPred2A-style)

> Dosztányi et al. (2005) *J. Mol. Biol.* **347**, 827–839  
> Mészáros et al. (2018) *Nucleic Acids Res.* **46**, W329–W337

**핵심 직관:** 잔기 $i$가 주변 잔기들과 강한 소수성·정전기 상호작용을 형성할 수 있다면 → 규칙 구조.  
형성하지 못한다면 → 무질서.

서열만으로 이 상호작용 에너지를 추정합니다.

---

### 7.2 단계 1: 20×20 쌍별 에너지 행렬

$$E_{\text{pair}}(i,j) = \underbrace{-\max(0, H_i) \cdot \max(0, H_j)}_{\text{소수성 기여}} + \underbrace{0.5 \cdot Q_i \cdot Q_j}_{\text{전하 기여}}$$

**소수성 지수 $H$** — Kyte-Doolittle 척도 (1982), 최댓값 4.5로 정규화:

| 아미노산 | KD 값 | $H = \text{KD}/4.5$ |
|---------|-------|--------------------|
| Ile (I) | 4.5 | 1.00 |
| Val (V) | 4.2 | 0.93 |
| Leu (L) | 3.8 | 0.84 |
| Ala (A) | 1.8 | 0.40 |
| Gly (G) | −0.4 | 0 (음수 클리핑) |
| Asp (D) | −3.5 | 0 (음수 클리핑) |
| Arg (R) | −4.5 | 0 (음수 클리핑) |

**형식 전하 $Q$** — pH 7 기준:
- Asp(D), Glu(E): $Q = -1$
- Lys(K), Arg(R): $Q = +1$
- 나머지: $Q = 0$

**행렬 해석:**
- Ile–Leu: $E = -1.00 \times 0.84 = -0.84$ → 유리 (규칙 선호)
- Asp–Asp: $E = 0 + 0.5 \times (-1)(-1) = +0.5$ → 불리 (무질서 선호)
- Asp–Lys: $E = 0 + 0.5 \times (-1)(+1) = -0.5$ → 유리 (염 다리, 규칙 선호)

---

### 7.3 단계 2: 슬라이딩 창 에너지 합산

잔기 $i$에 대해 ±10잔기 창 내의 모든 이웃 $j$와의 에너지 합산:

$$E_{\text{sum}}(i) = \sum_{\substack{j=i-10 \\ j \neq i}}^{i+10} E_{\text{pair}}(\text{aa}[i],\, \text{aa}[j])$$

- $E_{\text{sum}} < 0$ → 강한 소수성/정전기 상호작용 → **규칙 구조 선호**
- $E_{\text{sum}} \approx 0$ 또는 $> 0$ → 약한 상호작용 → **무질서 선호**

**복잡도:** $O(N \times 21)$ → 일반 단백질($N < 1000$)에서 무시할 수준

---

### 7.4 단계 3: 시그모이드 변환

에너지 합산값을 $[0, 1]$ 의 무질서 확률로 변환:

$$\text{score}(i) = \sigma(s \cdot E_{\text{sum}}(i) + b) = \frac{1}{1 + e^{-(s \cdot E_{\text{sum}}(i) + b)}}$$

- $s = 1.4$ (기울기): 에너지 차이에 대한 민감도
- $b = 0.7$ (편향): 중립 잔기($E_{\text{sum}} = 0$)도 약간 무질서 쪽으로 편향
  - $\sigma(0.7) \approx 0.67$ → 순수 Gly 서열의 기저 무질서 점수

**검증값:**

| 서열 | 예상 | score |
|------|------|-------|
| 순수 Ile | 강한 소수성 → 규칙 | ≈ 0.02 |
| 순수 Asp | 같은 부호 반발 → 무질서 | ≈ 0.97 |
| 순수 Gly | 상호작용 없음 → 중립 유연 | ≈ 0.62 |

**분류 기준:** score > 0.5 → 무질서 잔기

---

## 8. GPU 가속 — CUDA 구현

### 동기

단백질 에너지 계산의 핵심인 Born 반지름 계산과 쌍별 에너지 합산은 모두 $O(N^2)$ 연산입니다.  
GPU는 수천 개의 CUDA 코어를 이용해 이 쌍별 연산을 **병렬화**합니다.

---

### 8.1 아키텍처: CPU-GPU 하이브리드

```
[CPU]
  PDB 파싱, 이웃 목록 구성, SASA 계산,
  MC 증분 에너지(이동 원자만), Born 반지름 갱신

[GPU (CUDA)]
  전체 Born 반지름 계산 (born_sum_kernel + born_finalize_kernel)
  전체 쌍별 에너지 합산 (pair_energy_kernel)
  → 이웃 목록 재구성 시에만 호출
```

**이유:** MC 각 스텝에서 한 원자만 이동하므로, 이동된 원자의 이웃들만 재계산하는 증분 방식이 GPU 전체 계산보다 빠릅니다. GPU는 전체 재계산이 필요한 시점(이웃 목록 재구성, 에너지 기준점 갱신)에만 호출됩니다.

---

### 8.2 CUDA 커널 설명

#### Kernel 1: `born_sum_kernel`

```
입력: 이웃 목록 쌍 (i,j), 원자 위치, VDW 반지름
출력: 각 원자의 Born 합산값 sum[]

한 스레드 = 한 (i,j) 쌍
  → HCT 적분 I(i←j), I(j←i) 계산
  → atomicAdd로 sum[i], sum[j]에 누적
```

#### Kernel 2: `born_finalize_kernel`

```
입력: sum[], VDW 반지름 r[]
출력: 유효 Born 반지름 a[]

한 스레드 = 한 원자 i
  → a[i] = 1 / max(1/r[i] - 0.5*sum[i], 2)
```

#### Kernel 3: `pair_energy_kernel`

```
입력: 이웃 목록 쌍, 위치/전하/반지름/ε/Born 반지름
출력: 총 쌍별 에너지 (스칼라)

한 스레드 = 한 (i,j) 쌍
  → r > 12 Å 이면 스킵
  → DH + GB + LJ 에너지 계산 (단정밀도 float)
  → atomicAdd로 d_etot에 누적
```

**수치 정밀도:**  
GPU 커널은 단정밀도(float32)로 계산하고, CPU 증분 계산은 배정밀도(double64)를 사용합니다.  
단정밀도 오차는 일반적으로 ~0.01% 수준으로 MC 샘플링에 충분합니다.

---

### 8.3 OpenMP 병렬 MC 체인

```cpp
#pragma omp parallel
{
    // 각 스레드가 독립적인 난수 생성기를 가짐
    std::mt19937 lg(seed ^ thread_id);
    #pragma omp for schedule(dynamic)
    for (int c = 0; c < ncand; ++c)
        chain(c, lg);  // 독립 MC 궤적
}
```

여러 MC 체인을 OpenMP 스레드로 병렬 실행합니다. 각 체인은 독립적이므로 동기화가 필요 없습니다.

### Kabsch RMSD

두 구조 $\{\mathbf{p}_i\}$, $\{\mathbf{q}_i\}$ 사이의 최적 정렬 후 RMSD:

$$\text{RMSD} = \sqrt{\frac{1}{N}\sum_{i=1}^{N} |\mathbf{p}_i - R\mathbf{q}_i - \mathbf{t}|^2}$$

최적 회전 행렬 $R$은 SVD(특이값 분해)로 계산합니다:
$$\mathbf{H} = \mathbf{P}^\top \mathbf{Q} = U \Sigma V^\top \implies R = V U^\top$$

> Kabsch (1978) *Acta Cryst.* **A34**, 827–828
