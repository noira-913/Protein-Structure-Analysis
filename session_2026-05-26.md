# Session 2026-05-26 — P1.4a Dihedral Energy

## Completed

### P1.4a — Dihedral energy for all rotatable bonds (`src/physics_engine.cpp`)

**New structs / helpers** (all before `bond_templates`):
- `DihTerm { V2, n, gamma }` — one Fourier term
- `DihRecord { a, b, c, d, terms }` — 4-atom sequence + terms
- `get_dih_terms(kind, aname_j)` — parameter table:
  - `BACKBONE_PHI`: `{0.40,3,π}, {0.29,2,π}`
  - `BACKBONE_PSI`: `{0.40,3,0}, {0.29,2,π}`
  - `O`: `{1.18,2,π}, {0.14,3,0}` / `S`: `{0.60,3,0}` / `N`: `{0.50,3,0},{0.16,2,0}` / default C-C: `{0.56,3,0}`

**`BondTopology` additions:**
- `std::vector<DihRecord> dihedrals;` field
- Built in `build()` (after excl, before rot_bond_sides): one record per rot_bond; `a` = first adj[rb.i] ≠ rb.j, `d` = first adj[rb.j] ≠ rb.i; terminal bonds skipped
- `num_dihedrals` pybind11 read-only property

**`PhysicsEngine` private helpers:**
- `dihedral_angle(p, a,b,c,d)` — signed torsion via atan2(m·n2, n1·n2)
- `dihedral_e(p, dihs)` — full sum, used in `total_e()`
- `dihedral_e_boundary(p, dihs, in_side)` — sum only over records with ≥1 atom on each side of rotation; used in MC ΔE

**`total_e()`**: adds `dihedral_e(p, topo->dihedrals)` when `topo != nullptr`

**`generate_ensemble()` MC step ΔE**:
```
old_dih = dihedral_e_boundary(st, topo.dihedrals, in_side)  // before rotation
... rotate ...
new_dih = dihedral_e_boundary(st, topo.dihedrals, in_side)  // after rotation
dE = (new_cross - old_cross) + (new_sasa - old_sasa) + (new_dih - old_dih)
```
